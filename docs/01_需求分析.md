# 亚马逊竞品分析需求文档

## 项目背景
分析竞争链接的核心是找到对方的**核心优势、运营漏洞和流量逻辑**，从而反向优化自身链接、制定差异化策略。

## 核心关注维度

### 一、基础信息层 - 锚定竞品核心定位

**目标：** 精准匹配自身产品的竞品（避免跨价位/跨品类无效分析）

| 维度 | 关注项 | 分析要点 |
|------|--------|----------|
| **核心属性** | 类目节点 | 是否有高流量小众节点？ |
| | 产品规格/功能 | 主打卖点是否与自身重合？有无差异化功能？ |
| **价格体系** | 日常售价 | 定价区间 |
| | 促销价 | LD/BD价格策略 |
| | 捆绑销售价 | 组合定价 |
| | 变体价差 | 对方定价策略逻辑 |
| **变体布局** | 变体数量 | 全量分析 |
| | 变体维度 | 颜色/尺寸/功能哪个维度销量最高？ |
| | 热销变体 | 判断用户核心需求 |

### 二、流量与转化层 - 拆解竞品获客逻辑

**目标：** 找到竞品的核心流量入口，直接对应"对方的流量从哪来、为什么能转化"

#### 2.1 流量端：核心流量入口

**关键词布局：**

| 维度 | 分析项 |
|------|--------|
| 前台搜索 | 核心关键词自然排名（首页/前3页/前10名） |
| | 广告排名位置 |
| 链接内关键词 | 标题关键词布局 |
| | 五点描述关键词 |
| | 搜索词(ST)布局 |
| | A+页面核心关键词 |
| | 核心引流词 vs 长尾词分析 |

**广告与活动：**

| 维度 | 分析项 |
|------|--------|
| 广告表现 | 是否开广告（通过排名波动/前台广告位判断） |
| | 广告词和自然词占比 |
| 平台活动 | LD/BD/7天秒杀参与情况 |
| | 优惠券（金额/门槛） |
| | 满减活动 |
| | 活动频率和折扣力度 |
| | 活动运营节奏分析 |
