# 亚马逊竞品分析需求文档 v2.0

## 项目背景

作为亚马逊运营，分析竞争链接的核心是**找到对方的核心优势、运营漏洞和流量逻辑**，从而反向优化自身链接、制定差异化策略。

核心关注维度可分为**基础信息、流量转化、产品与运营、合规与风险**四大类，每类均提取可落地的核心关注项，避免无效信息堆砌。

---

## 一、基础信息：锚定竞品核心定位，快速分层对标

核心关注**精准匹配自身产品的竞品**（避免跨价位/跨品类无效分析），重点看：

| 维度 | 关注项 | 分析要点 |
|------|--------|----------|
| **核心属性** | 类目节点 | 是否有高流量小众节点 |
| | 产品规格/功能 | 主打卖点是否与自身重合，有无差异化功能 |
| **价格体系** | 日常售价 | 定价区间 |
| | 促销价 | LD/BD 价格策略 |
| | 捆绑销售价 | 组合定价 |
| | 变体价差 | 对方定价策略逻辑 |
| **变体布局** | 变体数量 | 全量分析 |
| | 变体维度 | 颜色/尺寸/功能哪个维度销量最高 |
| | 热销变体 | 判断用户核心需求 |

---

## 二、流量与转化：拆解竞品的核心获客逻辑，复制+优化

这是分析的**核心板块**，直接对应"对方的流量从哪来、为什么能转化"，是可直接借鉴的运营方向。

### 2.1 流量端：找到竞品的核心流量入口

#### 关键词布局

| 维度 | 分析项 |
|------|--------|
| **前台搜索** | 核心关键词的自然排名（首页/前3页/前10名） |
| | 广告排名位置 |
| **链接内关键词** | 标题关键词布局 |
| | 五点描述关键词 |
| | 搜索词(ST)布局 |
| | A+页面核心关键词 |
| | 核心引流词 vs 长尾词分析 |

#### 广告与活动

| 维度 | 分析项 |
|------|--------|
| **广告表现** | 是否开广告（通过排名波动/前台广告位判断） |
| | 广告词和自然词占比 |
| **平台活动** | LD/BD/7天秒杀参与情况 |
| | 优惠券（金额/门槛） |
| | 满减活动频率和折扣力度 |
| | 活动运营节奏分析 |

### 2.2 转化端：找到竞品的核心转化卖点

> 核心问题：用户为什么买他不买我？

#### 评价板块

| 分析项 | 关注点 |
|--------|--------|
| **核心数据** | 星级(4.0+/4.5+)、评价数量、日均增评数、好评率 |
| **好评分析** | 核心卖点（用户反复提及的点：材质、功能、性价比） |
| **差评分析** | 高频问题 → **对方的漏洞 = 自身的差异化机会** |

#### 链接内容

| 分析项 | 关注点 |
|--------|--------|
| **主图/副图/A+** | 主打场景是否贴合用户、卖点展示顺序、视频内容 |
| **五点描述** | 卖点是否清晰、是否解决用户痛点 |
| **产品详情** | 信息完整度、说服力 |

#### 销量与排名

| 分析项 | 关注点 |
|--------|--------|
| **BSR排名** | 排名波动（判断是否在推流/清库存） |
| **类目排名** | 大类目/子类目排名变化 |
| **变体销量** | 各变体销量占比 → 判断用户偏好 |

---

## 三、产品与运营：深挖竞品的供应链和运营策略，建立壁垒

结合自身供应链优势，找到竞品的可超越点：

| 维度 | 关注点 |
|------|--------|
| **产品迭代** | 链接是否有版本更新（新款升级功能、优化材质） |
| **供应链能力** | 发货方式(FBA/FBM/自发货)、库存深度、断货情况 |
| **运营节奏** | 新品期推流方式（测评+广告组合）、老品维护方式 |
| **品牌运营** | 店铺是否有社交媒体引流（Facebook/Instagram等站外流量） |

---

## 四、合规与风险：规避竞品的坑，降低自身运营风险

重点看竞品的**合规问题和潜在风险**，避免自身踩雷，同时找到反制机会：

| 风险类型 | 关注点 |
|----------|--------|
| **差评风险** | 高频差评是否为产品硬伤（设计缺陷、质量问题） |
| | 若为行业通病 → 针对性优化并作为自身卖点 |
| **平台处罚** | 排名是否突然下跌、评价是否被删除 |
| | 判断对方是否违规操作（刷评/刷单） |

---

## 核心分析技巧：精准对标 + 差异化突破

### 1. 分层对标

| 竞品类型 | 定义 | 分析重点 |
|----------|------|----------|
| **头部竞品** | BSR 前10 | 流量和运营策略 |
| **对标竞品** | 排名与自身相近 | 转化和价格 |
| **潜力竞品** | 新品但增长快 | 差异化卖点 |

### 2. 重点关注"差异点"

- ❌ 不要照搬竞品
- ✅ 找到对方的**未满足需求**
  - 差评中的高频痛点
  - 未覆盖的长尾关键词
- ✅ 作为自身的核心差异化卖点

### 3. 定期跟踪

- 每周/每两周更新竞品数据
- 重点关注：排名、价格、评价的波动
- 及时调整自身运营策略

---

## 核心逻辑

所有分析最终都要落地到**自身链接的优化**：

```
┌─────────────────────────────────────────────────────────────┐
│  复制竞品的有效策略（关键词布局、活动节奏）                    │
│  +                                                          │
│  规避竞品的漏洞（差评痛点、合规风险）                         │
│  +                                                          │
│  放大自身的优势（供应链、差异化功能）                         │
│  =                                                          │
│  流量比对方多 · 转化比对方高 · 产品比对方优                   │
└─────────────────────────────────────────────────────────────┘
```
