## 二、数据采集方案

### 2.1 方案对比与选型

| 方案 | 数据覆盖 | 成本 | 合规性 | 可靠性 | 推荐度 |
|------|---------|------|--------|--------|--------|
| **PA-API 5.0** | 基础信息、当前价格、BSR | 免费(需Associates账号) | ✅ 官方API | 高 | ⭐⭐⭐⭐ |
| **Keepa API** | 历史价格、历史BSR、销量估算 | $19-49/月 | ✅ 合规 | 高 | ⭐⭐⭐⭐⭐ |
| **Rainforest API** | 完整页面数据、评论全文、搜索排名 | $75-200/月 | ✅ 第三方服务 | 高 | ⭐⭐⭐⭐ |
| **ScraperAPI** | 页面数据(需自行解析) | $49/月起 | ⚠️ 灰色地带 | 中 | ⭐⭐⭐ |
| **Playwright 爬虫** | 所有前台可见数据 | 代理费用 | ❌ 违反ToS | 中 | ⭐⭐(备用) |
| **Helium10/JungleScout** | 关键词数据、搜索量 | $99-279/月 | ✅ SaaS服务 | 高 | ⭐⭐⭐⭐ |

### 2.2 推荐方案：三层数据采集架构

```
优先级 1 (首选): PA-API 5.0
  → 合法、免费、获取商品基础信息和当前价格
  → 限制: 无历史数据、无关键词数据、无评论全文

优先级 2 (补充): Keepa API + Rainforest API
  → Keepa: 历史价格/BSR趋势，是核心竞品分析数据源
  → Rainforest: 评论全文、完整listing内容、搜索排名

优先级 3 (备用): Playwright 无头浏览器
  → 当API无法获取的数据 (A+内容、变体详情)
  → 配合 playwright-stealth 反检测
  → 住宅代理轮换
```

### 2.3 PA-API 5.0 详细说明

**可获取数据：**
- 商品标题、图片、品牌、分类
- 当前价格 (日常价、促销价)
- BSR 排名 (当前快照)
- 评分 (平均分 + 评价数)
- 变体信息 (父ASIN、子ASIN列表)

**限制：**
- 需要 Amazon Associates 账号 (180天内需产生3笔销售)
- 速率限制: 1 TPS (每秒1次请求)，每日上限 8,640 次
- 无历史数据、无关键词数据、无评论全文、无A+内容

### 2.4 Keepa API

**核心价值：** 业界最全面的亚马逊历史数据库，追踪超过18亿商品。

**可获取数据：**
- 历史价格曲线 (Amazon自营、第三方新品、二手、Buy Box)
- 历史BSR走势
- 历史评论数/评分变化
- 优惠券/Deal检测
- 变体数据、FBA费用数据

**定价：** Token制，$19-49/月起，按调用复杂度消耗Token。

### 2.5 Playwright 爬虫 (备用方案)

当API无法覆盖时，使用 Playwright 无头浏览器作为备用：

**关键技术点：**
- 使用 `playwright-stealth` 绕过自动化检测
- 屏蔽图片/CSS/字体加载，提速40-60%
- 住宅代理轮换 (SmartProxy/Bright Data)
- 请求间隔 3-5 秒，避免触发反爬

**核心选择器 (产品详情页)：**

| 数据 | CSS 选择器 |
|------|-----------|
| ASIN | `#ASIN` (hidden input) |
| 标题 | `#productTitle` |
| 价格 | `#corePrice_feature_div .a-offscreen` |
| 评分 | `#acrPopover span.a-icon-alt` |
| 评价数 | `#acrCustomerReviewCount` |
| BSR | `#productDetails_detailBullets_sections1` 中的 "Best Sellers Rank" 行 |
| 五点描述 | `#feature-bullets ul li span.a-list-item` |
| 品牌 | `#bylineInfo` |
| A+内容 | `#aplus` |
| 变体 | `#twister` (变体选择器容器) |

## 三、技术栈选型

### 3.1 核心技术栈

| 层级 | 技术选择 | 理由 |
|------|---------|------|
| **语言** | Python 3.11+ | 生态丰富、开发效率高 |
| **HTTP 客户端** | httpx (通用) / curl_cffi (反检测) | httpx 支持异步+HTTP/2；curl_cffi 可模拟浏览器TLS指纹 |
| **HTML 解析** | parsel (首选) / BeautifulSoup+lxml (备选) | parsel 快速、支持CSS+XPath，Scrapy生态 |
| **浏览器自动化** | Playwright + playwright-stealth | 比Selenium更快、更隐蔽、原生异步 |
| **数据处理** | pandas + numpy | 数据分析标配 |
| **数据存储** | SQLite (原型阶段) | 零配置、文件级、足够竞品分析规模 |
| **交互仪表盘** | Streamlit | 最快实现路径、Python原生、内置缓存 |
| **图表库** | Plotly (交互) + Matplotlib/Seaborn (静态) | Plotly一套代码兼容仪表盘和HTML报告 |
| **报告生成** | Jinja2 + Plotly.to_html() (HTML) / WeasyPrint (PDF) | 自包含HTML报告 + PDF导出 |
| **情感分析** | TextBlob / VADER | 轻量级、无需GPU、足够评论级分析 |
| **配置管理** | PyYAML | 简洁的YAML配置文件 |

## 四、数据模型设计

### 4.1 数据库表结构

```sql
-- 商品主表
CREATE TABLE products (
    asin TEXT PRIMARY KEY,
    title TEXT,
    brand TEXT,
    category_path TEXT,
    bullet_points JSON,
    image_urls JSON,
    variant_asins JSON,
    parent_asin TEXT,
    seller_name TEXT,
    is_fba BOOLEAN,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 价格/BSR 历史追踪表
CREATE TABLE price_history (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    asin TEXT REFERENCES products(asin),
    price REAL,
    currency TEXT DEFAULT 'USD',
    bsr_rank INTEGER,
    bsr_category TEXT,
    rating_average REAL,
    rating_count INTEGER,
    availability TEXT,
    recorded_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 关键词追踪表
CREATE TABLE keyword_tracking (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    keyword TEXT NOT NULL,
    asin TEXT REFERENCES products(asin),
    organic_rank INTEGER,
    sponsored_rank INTEGER,
    page_number INTEGER,
    recorded_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 评论数据表
CREATE TABLE reviews (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    asin TEXT REFERENCES products(asin),
    review_id TEXT UNIQUE,
    rating INTEGER,
    title TEXT,
    body TEXT,
    author TEXT,
    review_date DATE,
    verified_purchase BOOLEAN,
    sentiment_score REAL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 索引
CREATE INDEX idx_price_history_asin ON price_history(asin);
CREATE INDEX idx_price_history_date ON price_history(recorded_at);
CREATE INDEX idx_keyword_tracking_keyword ON keyword_tracking(keyword);
CREATE INDEX idx_keyword_tracking_asin ON keyword_tracking(asin);
CREATE INDEX idx_reviews_asin ON reviews(asin);
```

## 五、可视化方案

### 5.1 图表类型与适用场景

| 分析需求 | 图表类型 | 实现方案 |
|---------|---------|---------|
| 价格对比 | 分组柱状图 / 带范围的折线图 | `plotly.go.Bar` (grouped) / `go.Scatter` (fill) |
| BSR 趋势 | 多线时序图 (对数Y轴, 倒序) | `plotly.go.Scatter` (yaxis_type="log") |
| 关键词排名热力图 | 矩阵热力图 (商品×关键词) | `plotly.go.Heatmap` / `seaborn.heatmap` |
| 评论情感分析 | 堆叠柱状图 + 词云 | `plotly.go.Bar` (stacked) + `wordcloud` |
| 市场份额 | 环形图 / 树状图 | `plotly.go.Pie` (hole=0.4) / `go.Treemap` |
| 流量来源 | 桑基图 / 堆叠面积图 | `plotly.go.Sankey` |
| 价格历史 | 时间线折线图 + 事件标注 | `plotly.go.Scatter` + annotations |

### 5.2 报告输出格式

1. **交互式HTML报告** (主推)：Plotly图表 + Jinja2模板，单文件可分享
2. **PDF报告**：WeasyPrint 将HTML转PDF，图表用 kaleido 导出静态图
3. **Excel数据导出**：openpyxl 导出原始分析数据

## 九、合规性说明

1. **优先使用官方API** (PA-API 5.0, SP-API)，合法合规
2. **第三方API** (Keepa, Rainforest) 作为合规的商业数据服务
3. **Playwright 爬虫仅作备用**，遵循以下原则:
   - 请求间隔 ≥ 3 秒
   - 遵守 robots.txt
   - 不采集个人信息
   - 缓存响应，24小时内不重复抓取
4. 亚马逊 ToS 明确禁止自动化数据采集，爬虫方案存在法律风险
