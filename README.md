## 当前任务：竞品分析框架

### 一、基础信息层 - 锚定竞品核心定位

| 维度 | 核心关注项 |
|------|-----------|
| 核心属性 | 类目节点(高流量小众节点)、产品规格/功能、主打卖点 |
| 价格体系 | 日常售价、促销价(LD/BD)、捆绑价、变体价差 |
| 变体布局 | 变体数量、变体维度(颜色/尺寸/功能)、热销变体分析 |

### 二、流量与转化层 - 拆解获客逻辑

#### 2.1 流量端：核心流量入口

**关键词布局：**
- 前台搜索：核心词自然排名(首页/前3页/前10)、广告排名
- 链接内关键词：标题、五点描述、ST、A+页面关键词

**广告与活动：**
- 广告表现：是否开广告、关键词排名波动
- 平台活动：LD/BD/7天秒杀、优惠券、满减活动频率和力度
